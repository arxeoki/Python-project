To run this project, you need:

## Installation and Running the Project

Follow the steps below to set up and execute the project on your machine.

### 1. Clone the repository

git clone https://github.com/arxeoki/Python-project.git
cd Python-project

### 2. Install required dependencies

pip install -r requirements.txt

### 3. Run the project

python src/main.py
